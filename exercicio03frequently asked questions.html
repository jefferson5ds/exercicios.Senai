Questionario

1 – O que significa a sigla HTML?

R: HyperText Markup Language — Linguagem de Marcação de Hipertexto


2 – Que tipo de linguagem é o HTML?

R: Linguagem de marcação, ou seja, serve para estruturar informações e criar páginas web


3– Para qual função a tag h1 é utilizada?

R: Para acrescentar título apresentado com o maior tamanho de texto


4 – Quais tags são usadas para inserir respectivamente uma linha horizontal, uma imagem e um link?

R: hr, img e a


5 – Qual a função do atributo alt na tag ?

R: É utilizado para fornecer uma descrição da imagem


6 – Qual a função da tag em uma tabela?

R: TD significa "Table Data" e define cada uma das células da tabela.
